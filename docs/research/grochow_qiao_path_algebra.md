# Path algebra `F[Q_G] / J²` (R-TI Layer T0.1)

**Status.** Paper synthesis deliverable for Workstream R-TI Layer T0
(audit plan § "Decision GQ-D"); landed as a single commit alongside
the other Layer T0 documents. No Lean implementation depends on this
file directly — it is a one-page reference for the
post-Decision-GQ-A encoder choice that drives Layers T1–T6.

## 1. Quiver definition

For a directed graph `G = (V, E)` with `V = {0, …, m-1}` and
`E ⊆ V × V` (we treat `E` as a `Fin m → Fin m → Bool` adjacency
predicate; an undirected graph is the special case where `adj` is
symmetric), the **quiver `Q_G`** is the directed multigraph with:

* **Vertices:** `V = {0, …, m-1}`;
* **Arrows:** for each ordered pair `(u, v)` with `adj u v = true`,
  one arrow `α(u, v) : u → v`. Self-loops permitted when
  `adj u u = true`.

Quivers are formalised in Mathlib as the typeclass
`Quiver.{v} (V : Type u)` with `Hom : V → V → Type v`. For us `V :=
Fin m` and `Hom u v := { _ : Unit // adj u v = true }` (a single arrow
when present, none otherwise) — but we do not need the full
`Quiver`-typeclass infrastructure: an explicit `inductive QuiverArrow`
type suffices, as documented in Layer T1's Sub-task T1.1.

## 2. Path algebra `F[Q_G]`

The path algebra `F[Q_G]` is the F-vector space spanned by all
finite-length paths in `Q_G` (including length-zero "lazy" paths
`e_v`, one per vertex), equipped with multiplication by path
composition (where composable, else zero):

```
e_v · e_v   = e_v
e_v · α(v, w) = α(v, w)             when v matches the source of α
α(u, v) · e_v = α(u, v)             when v matches the target of α
α(u, v) · α(v, w) = α(u, v, w)      length-2 path (would be a basis element)
α(u, v) · α(v', w) = 0              when v ≠ v'
```

`F[Q_G]` is **infinite-dimensional** in general (paths of any length
contribute basis elements), so it is not directly suitable as a
finite-dimensional encoder. We use the **radical-2 quotient** instead.

## 3. Radical-2 quotient `F[Q_G] / J²`

Let `J = rad(F[Q_G])` be the **Jacobson radical** of the path
algebra: in our case, `J` is the (two-sided) ideal generated by the
length-≥-1 paths (i.e., the span of all arrows). Then:

* `J²` is the span of length-≥-2 paths.
* `F[Q_G] / J²` has explicit basis `{e_v : v ∈ V} ∪ {α(u, v) :
  adj u v = true}`.
* **Dimension:** `m + |E_directed|`. For our `Fin m → Fin m → Bool`
  adjacency model, `|E_directed| = (Finset.univ.filter (fun p =>
  adj p.1 p.2)).card`.
* The quotient is **finite-dimensional** for every `m`, suitable as
  a Lean `Algebra` instance over `ℚ`.
* The radical-2 truncation **preserves vertex distinguishability**:
  every vertex idempotent `e_v` survives the quotient, and they
  remain pairwise orthogonal primitive idempotents (`e_v · e_w =
  δ_{v,w} e_v`).

## 4. Vertex-idempotent uniqueness

The cryptographically essential property of `F[Q_G] / J²` is that
its **vertex idempotents are recoverable from the algebra
structure alone**: an element `b ∈ F[Q_G] / J²` is a *primitive*
idempotent if and only if `b = e_v` for some `v ∈ V`.

**Sketch.** Any element of `F[Q_G] / J²` decomposes uniquely as a
sum `b = ∑_v λ_v e_v + ∑_{u,v ∈ E} μ_{u,v} α(u, v)`. Squaring:

```
b² = ∑_v λ_v² e_v +
     ∑_{u,v ∈ E} (λ_u μ_{u,v} + μ_{u,v} λ_v) α(u, v) +
     (length-2 contributions, killed by quotient)
   = ∑_v λ_v² e_v +
     ∑_{u,v ∈ E} (λ_u + λ_v) μ_{u,v} α(u, v)
```

Setting `b² = b` and matching basis coefficients:

* `λ_v² = λ_v` ⇒ `λ_v ∈ {0, 1}` (over a field of characteristic
  zero; the same holds over any field).
* `(λ_u + λ_v) μ_{u,v} = μ_{u,v}` ⇒ either `μ_{u,v} = 0` or
  `λ_u + λ_v = 1`.

If `b` is a *primitive* idempotent (not a non-trivial sum of two
orthogonal idempotents), exactly one `λ_v` is `1`, all other
`λ_w = 0` (`w ≠ v`), and every `μ_{u,v}` with the chosen `λ_v = 1`
is forced to zero (the "λ_u + λ_v = 1" condition picks one endpoint
having `λ = 0`, the other having `λ = 1`, but then summing two
primitive-on-`v` idempotents would split `b`). The conclusion:
`b = e_v` for that distinguished `v`.

This argument is the path-algebra version of a standard fact in
representation theory of artinian algebras (Auslander–Reiten–Smalø,
*Representation Theory of Artin Algebras*, §III.2). In Lean, the
forward direction is what Sub-task T1.6 formalises:

```lean
theorem pathAlgebra_idempotent_iff_vertex
    (m : ℕ) (adj : Fin m → Fin m → Bool)
    (i : Fin (pathAlgebraDim m adj)) :
    -- b_i is idempotent iff b_i is a vertex idempotent.
    (b_i · b_i = b_i) ↔ ∃ v, b_i = e_v
```

The *primitivity* refinement (idempotent and not a non-trivial sum)
is Sub-task T4.3. For the iff in our reduction, primitivity is
needed only to identify the σ-action on idempotents in the reverse
direction (T4.3 → T4.4); the forward direction (T3) doesn't depend
on it.

## 5. Why this choice over `F[A_G]` (adjacency algebra)

The pre-audit plan proposed the **commutative adjacency algebra**
`F[A_G] = F[A] / (minpoly_A)` where `A ∈ Mat(m, F)` is the adjacency
matrix. This choice has a **proven soundness defect**:

* **Cospectral non-isomorphic graphs** (e.g., the Saltire pair on
  5 vertices, or any pair of strongly regular cospectral graphs)
  have isomorphic `F[A_G]` (because they share `A`'s minimal
  polynomial) but non-isomorphic `Aut` groups. The iff conclusion
  in our Karp reduction would *fail in the reverse direction* on
  such pairs.

The **path algebra `F[Q_G] / J²`** repairs this defect: distinct
graphs (even cospectral pairs) yield distinct path algebras because
the *quiver structure* (not just spectrum) is encoded. Specifically:

* The **set of arrows** `{(u, v) : adj u v = true}` is recoverable
  from `F[Q_G] / J²` via the basis-classification lemma above —
  arrow basis elements are characterised as elements `b` with
  `b² = 0` and `e_u · b · e_v = b` for unique `(u, v)`.
* A graph automorphism σ ∈ Aut(G) corresponds *bijectively* with an
  algebra automorphism of `F[Q_G] / J²`; the algebra-Aut group
  equals the graph-Aut group with no spectral restriction.

This bijection is the **rigidity property** the rigidity argument
(Layer T5) consumes: the GL³-orbit of `F[Q_G] / J²`'s structure
tensor determines the graph up to isomorphism, with no
cospectral-pair false equivalences.

## 7. Self-audit

**Finite-dimensionality.** `F[Q_G] / J²` has basis `{e_v} ∪
{α(u, v) : adj u v = true}` of cardinality `m + |E|`, which is
bounded by `m + m * m = m + m²`. ✅ Finite-dimensional.

**Vertex distinguishability.** The vertex-idempotent uniqueness
sketch above shows distinct vertex idempotents `e_v`, `e_w` (for
`v ≠ w`) are distinguishable by their unique support
`Subtype.val (pathArrowEquiv⁻¹ (e_v))`. ✅ Distinguishable.

**Iso-class faithfulness.** The path-algebra-Aut bijection follows
from the Auslander-Reiten-Smalø reference (Theorem III.2.1). ✅ Iso-
class faithful.

All three properties confirmed; **encoder choice GQ-A is sound for
the planned reduction.**
